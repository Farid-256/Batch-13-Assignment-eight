# QurbaniHat

## Project Overview:
QurbaniHat is a fully responsive web application designed to help users browse, explore, and book healthy Qurbani animals (Cows, Goats, and Sheep) online. It provides a clean and intuitive interface where users can view animal details, filter and sort by price, and confirm bookings with ease.
The project follows best practices with a well-structured codebase, meaningful Git commits, secure authentication, and smooth performance across all devices.

## Live URL:
[https://batch-13-assignment-eight.vercel.app](https://batch-13-assignment-eight.vercel.app)

## Technologies Used:
1. Next.js
2. Tailwind CSS
3. Better Auth
4. MongoDB
5. React Hook Form
6. React Toastify
7. Animate.css

## Features:
- Fully responsive design (mobile, tablet, and desktop)
- Browse all Qurbani animals with sort by price (Low → High / High → Low)
- Detailed animal page with full information
- Booking form with toast success notification
- Secure authentication with Email/Password and Google Sign-In
- Private routes — only logged-in users can access animal details and profile
- User profile page with name, photo, and email
- Update profile feature (name and photo)
- Banner section with Animate.css animations
- Qurbani Tips and Top Breeds sections
- Custom 404 Not Found page
- Error-free deployment on Vercel
